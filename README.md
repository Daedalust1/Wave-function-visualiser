# Wave-function-visualiser
My first project, it's a simple code to visualize the wave function of a particle in a 2D infinite square box by numerically solving the 2D Time Independent Schrodinger Equation.
The file consists of a singular function called "wave_function_plotter" which contains two inputs.
The first input determines the resolution of the image by increasing the size of the grid, a value of about 100-120 gives both a fast and a clear result.
The second input determines the number of states for which the graph is plotted this can be of any value but I have only plotted it upto 8 states.
